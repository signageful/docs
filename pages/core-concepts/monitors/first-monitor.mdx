# Erstellen des ersten Monitors

Das Erstellen des ersten Monitors ist einfach und schnell. Folgen Sie diesen Schritten, um Ihren ersten Monitor zu erstellen:

## Öffnen Sie die Übersichtsseite für Monitore

Wählen Sie im linken Navigationsmenü den Punkt "Monitore" aus, um zur Übersichtsseite für Monitore zu gelangen.

## Erstellen Sie einen neuen Monitor:

Klicken Sie auf den Button `Add Monitor`, um ein Dialogfenster zu öffnen, in dem Sie den Namen des Monitors und den Code eingeben können. _**Verwenden Sie einen eindeutigen und leicht identifizierbaren Namen für den Monitor**_.

### Geben Sie den Code ein:

Geben Sie einen Code ein, der auf dem jeweiligen Monitor angezeigt wird und zur Verifikation dient.

### Bestätigen Sie

Bestätigen Sie die Eingabe, um den Monitor zu erstellen.
