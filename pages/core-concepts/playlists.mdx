# Einführung in Wiedergabelisten

Eine Wiedergabeliste ist eine Sammlung von Medieninhalten, wie Videos oder Musik, die in einer bestimmten Reihenfolge und für eine bestimmte Dauer abgespielt werden können. Wiedergabelisten können verwendet werden, um verschiedene Arten von Inhalten zusammenzufassen und auf einem oder mehreren Monitoren anzuzeigen.

In diesem Leitfaden werden wir die Grundlagen des Erstellens und Verwalten von Wiedergabelisten behandeln, einschließlich:

- Hinzufügen und Entfernen von Medieninhalten
- Einstellen der Wiedergabereihenfolge und Dauer jedes Elements
- Zuweisen von Wiedergabelisten zu bestimmten Monitoren

Am Ende dieses Leitfadens haben Sie ein besseres Verständnis dafür, wie Sie Wiedergabelisten erstellen und verwalten können, um Ihre Mediendarstellung zu verbessern.
