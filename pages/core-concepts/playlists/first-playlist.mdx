# Erstellen der ersten Playlist

Nachdem Sie Ihren ersten Content erstellt haben, können Sie jetzt mit der Erstellung Ihrer ersten Wiedergabeliste fortfahren. Folgen Sie diesen Schritten, um Ihre erste Playlist zu erstellen:

## Öffnen Sie die Übersichtsseite für Playlisten:
Wählen Sie im linken Navigationsmenü den Punkt "Playlist" aus, um zur Übersichtsseite für Playlisten zu gelangen.

## Erstellen Sie eine neue Playlist:
Klicken Sie auf den Button `+ Add Playlist`, um ein Modal/Dialogfenster zu öffnen, in dem Sie den Namen der Playlist eingeben und mit `Add` bestätigen können.

## Bearbeiten Sie die Playlist:

Auf der Bearbeitungsseite der Playlist stehen Ihnen verschiedene Optionen zur Verfügung. 

### Basic 
Im Bereich `Basics` können Sie den Namen der Playlist bearbeiten.

### Monitors 
Im Bereich `Monitore` können Sie Monitore hinzufügen oder entfernen.

### Content

Im Bereich `Content` können Sie den hinzugefügten Content in die gewünschte Reihenfolge bringen durch die Drag and Drop Funktionalität und für jeden hinzugefügten Content die Laufzeit definieren. Wenn die Duration auf `0` steht, läuft der Content unendlich.
